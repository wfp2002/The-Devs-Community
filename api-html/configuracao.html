Configuração



Abaixo voce confere os primeiros passos para utilização das nossas APIs:

1. Criação de conta

Para ter acesso aos recursos disponibilizados pela API, é necessário possuir uma conta de desenvolvedor no Portal de Desenvolvedores da Ambev.

2. Criação de APPs

Todas as chamadas à API são feitas por meio de APPs configuradas pelo desenvolvedor em sua conta. A APP possui duas informações utilizadas para chamadas das APIs, que são o CLIENT_ID e o ACCESS_TOKEN. Identificadores únicos gerados para cada APP e que irão identificar a orígem das chamadas.

3. Conhecendo o ambiente Sandbox

Agora que você possui uma APP e suas respectivas chaves de acesso, já é possível iniciar as integrações com a API da Ambev. Para facilitar seus testes, a Ambev disponibiliza um ambiente de Sandbox, com as mesmas operações disponíveis no ambiente de produção.
